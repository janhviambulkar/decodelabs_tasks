Find Your Dream Internship & Job

CareerConnect is a modern Full Stack Job Portal built using HTML, CSS, JavaScript, Node.js and Express.js. It helps students discover internships and full-time opportunities through a fast, responsive and user-friendly interface.

Project Overview

CareerConnect is a responsive internship and placement portal developed as the DecodeLabs Full Stack Internship Final Project. The application demonstrates frontend development, REST API integration, backend development using Express.js, GitHub version control, and deployment using Netlify and Render.
