## 📌 Overview

**CareerConnect** is a modern internship and job discovery platform designed to help students and fresh graduates explore career opportunities from leading companies.

The platform provides a clean, user-friendly interface where users can discover internships, search for roles based on skills, and explore hiring companies.

This project is developed as part of my Full Stack Development learning journey, focusing on building real-world web applications with scalable frontend and backend architecture.
