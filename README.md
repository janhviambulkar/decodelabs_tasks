# DecodeLabs Full Stack Student Portal

## 📌 Overview
This project was developed as part of the DecodeLabs Full Stack Development Internship.

It demonstrates frontend and backend integration using HTML, CSS, JavaScript, Node.js, and Express.js.

## 🚀 Features
- Responsive User Interface
- Student Information Display
- REST API Integration
- Express.js Backend
- Modern Landing Page

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js
